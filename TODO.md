# Taskboard 優先度機能の実装 TODO

現在の `taskboard.js` には内部的に `priority` プロパティが存在しますが、UIから操作・表示できません。これを実用的な機能として拡張します。

## 1. 優先度の定義と表示 (Visuals)
- [x] 優先度ごとのカラーパレット定義 (CSS変数)
    - P1 (High): レッド / 緊急
    - P2 (Medium): イエロー / 通常 (デフォルト)
    - P3 (Low): ブルー / 低
- [x] タスクカード上に優先度バッジを表示する
    - `.cardTop` 内に表示
    - 優先度に応じた色分け (CSSクラス)

## 2. 優先度の編集機能 (Actions)
- [x] タスク作成時に優先度を選択できるようにする
    - 詳細ダイアログ (`<dialog>`) を導入し、新規作成・編集時に優先度を選択可能に。
- [x] 既存タスクの優先度を変更するボタンを追加
    - `Priority` ボタンによるトグル、および詳細ダイアログからの編集の両方に対応。

## 3. データ永続化と履歴 (Persistence)
- [x] 優先度変更時に `tasks.ndjson` に保存する
- [x] 優先度変更イベントを `events.ndjson` に記録する
    - `type: "priority_changed"`

## 4. 応用機能 (Advanced) - 任意
- [x] 優先度によるフィルタリング機能
    - 画面上部に「P1のみ表示」などのフィルタボタンを配置
- [ ] 優先度順のソート
    - カラム内で優先度が高いものを上に表示するオプション

## 5. 期限管理機能 (Due Date Management) - NEW
- [ ] タスクに期限 (`due_date`) フィールドを追加
    - 詳細ダイアログに `<input type="date">` を設置。
    - `tasks.ndjson` への保存と `events.ndjson` へのイベント記録。
- [ ] タスクカードへの期限表示 (Visuals)
    - 期限が設定されている場合、カード上に日付を表示。
    - **期限切れ (Overdue)**: 日付を赤色で強調表示。
    - **期限間近 (Upcoming)**: 期限当日や前日に警告色を表示。
- [ ] 期限によるソート/フィルタリング (Optional)
    - 期限が近い順に並べ替える機能。
