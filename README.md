# Taskboard

これは、ローカルファイルシステムで直接動作する、シンプルなサーバーレスのカンバンボードアプリケーションです。ブラウザのFile System Access APIを利用して、手元のプロジェクトフォルダ内にタスク管理用の`.taskboard`ディレクトリを作成し、タスクを管理します。

## 主な機能

- **ローカルファースト:** サーバーは不要です。すべてのデータはローカルのフォルダに保存されます。
- **プロジェクト連携:** 既存のプロジェクトフォルダを選択し、その中にタスクボードを作成できます。
- **基本的なカンバン機能:**
    - タスクの作成、名前の変更、削除
    - ドラッグ＆ドロップによるタスクの移動と並べ替え
    - タスクごとのチェックリスト機能
- **データの永続化:** すべての変更は、選択したフォルダ内の`.taskboard`ディレクトリに自動的に保存されます。
- **データ形式:** タスクデータは、AIや他のツールが解釈しやすいように、人間が読めるNDJSON（Newline Delimited JSON）形式で保存されます。

## 使い方

1.  リポジトリをクローンまたはダウンロードします。
2.  Pythonがインストールされている環境で、以下のコマンドを実行します。
    ```bash
    python start.py
    ```
3.  スクリプトが自動的にブラウザで`taskboard.html`を開きます。
4.  **"Open Project Folder"** ボタンをクリックし、タスク管理を行いたい任意のフォルダ（例: あなたのソフトウェアプロジェクトのルートフォルダ）を選択します。
5.  **"Initialize .taskboard"** ボタンをクリックすると、選択したフォルダ内に`.taskboard`ディレクトリが作成され、ボードが利用可能になります。
6.  あとはタスクを追加して、自由に管理しましょう！

## 動作の仕組み

このアプリケーションは、モダンブラウザに搭載されている **File System Access API** を活用しています。これにより、ユーザーが明示的に許可したフォルダに対して、ブラウザ上のJavaScriptが直接ファイルの読み書きを行えるようになります。

`start.py`は、HTML、CSS、JavaScriptファイルをブラウザに提供するための簡易的なローカルWebサーバーとして機能するだけです。アプリケーションのコアロジックとデータ管理は、すべてブラウザ側で完結しています。

## プロジェクトのファイル構成

```
taskboard/
├── .gitignore
├── README.md         <-- このファイル
├── start.py
├── taskboard.css
├── taskboard.html
├── taskboard.js
└── (任意のあなたのプロジェクトフォルダ)/
    └── .taskboard/   <-- 初期化後に自動作成される
        ├── board.json
        ├── events.ndjson
        ├── README.md
        └── tasks.ndjson
```

-   `taskboard.html`: アプリケーションの基本的な骨格を定義するHTMLファイル。
-   `taskboard.css`: アプリケーションのスタイルを定義するCSSファイル。
-   `taskboard.js`: File System Access APIを使ったファイルの読み書き、タスクの管理、UIの操作など、アプリケーションのすべてのロジックを含むJavaScriptファイル。
-   `start.py`: `http.server`を使ってローカルWebサーバーを起動し、ブラウザでアプリケーションを開くためのシンプルなPythonスクリプト。
-   `.gitignore`: Gitで無視するファイルを指定します。

## `.taskboard` ディレクトリ

ユーザーが選択したプロジェクトフォルダ内には、以下のファイルを含む`.taskboard`ディレクトリが作成されます。

-   `board.json`: カンバンボードのカラム設定（例: "To Do", "Doing", "Done"）などを保存します。
-   `tasks.ndjson`: 現在のすべてのタスクのスナップショット。1行に1つのタスクがJSON形式で記録されます。
-   `events.ndjson`: タスクの作成、移動、削除といったすべての操作の履歴が追記されていくイベントログ。
-   `README.md`: `.taskboard`ディレクトリ内のデータ形式について説明するファイル。
